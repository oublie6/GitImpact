# 当前能力与演进路线

## 已实现能力

- 基于 JWT 的基础认证
- `config` / `db` / `mixed` 三种认证模式
- 仓库记录维护与手动抓取
- 分析任务创建、异步执行、日志与报告保存
- OpenCode CLI 集成
- Markdown 和结构化 JSON 双报告
- MySQL / 达梦初始化脚本
- 基础前端原型
- vendor 模式构建与 Docker 构建

## 当前限制

- `ServerAnalyzer` 未实现
- 任务队列不是独立服务
- 并发任务会共享仓库缓存
- 前端仍偏原型，不是完整控制台
- 任务重试、取消、重新执行能力缺失

## 建议的后续扩展方向

- 引入独立 worker 和任务队列
- 为前端补齐表单、表格和错误提示
- 为任务执行过程增加更细粒度日志
- 给结构化 JSON 增加 schema 校验
- 支持报告导出历史与归档策略
